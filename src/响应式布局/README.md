# 响应式布局

## 响应式是什么

简而言之：一套代码，兼容多个终端。

## rem 实现响应式布局

在 [CSS 中长度单位与 px 转 rem](../CSS中长度单位与px转rem/README.md) 详细介绍了 rem 是什么。

同时，rem 在响应式中有极大的应用。

通过 CSS3 中的 `@media` 来对不同的宽度设置不同的 `font-size`, 随后使用 `rem` 实现不同的大小。

但是，rem 也有一定的问题：例如，root 处的 `font-size` 应用到全局并不太全实用。

## 媒体查询实现响应式布局

使用 `@media` 来针对不同大小的屏幕来实现不同的样式。
